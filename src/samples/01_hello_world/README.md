# サンプル01：はじめてのコンポーネント（Hello World）

## 何を学ぶか

- **JSX** の基本的な書き方
- **関数コンポーネント** の定義方法
- コンポーネントを **入れ子（ネスト）** にする方法

---

## コードの解説

### コンポーネントとは？

React では UI の部品を「コンポーネント」と呼びます。  
コンポーネントは **JSX を返す関数** です。

```tsx
function Greeting() {
  return <p>こんにちは！</p>
}
```

### コンポーネントの命名ルール

- 先頭は **大文字** にしなければなりません（例：`Greeting`、`MyButton`）
- 小文字だと React は HTML タグとして解釈してしまいます

### JSX とは？

HTML のように見えますが、実は JavaScript を書きやすくしたものです。  
JavaScript の式を `{}` で埋め込むことができます。

```tsx
const name = '太郎'
return <p>こんにちは、{name}さん！</p>
```

### コンポーネントを組み合わせる

コンポーネントは HTML タグのように入れ子にできます。

```tsx
function Parent() {
  return (
    <div>
      <Child />   {/* 子コンポーネントを呼び出す */}
    </div>
  )
}
```
